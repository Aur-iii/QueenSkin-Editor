QueenSkin Editor Help

QueenSkin Editor was designed out of a lack of good skin editors. It is built to be precise, user-friendly, and fun to work in.

This app is currently in its community phase. Feedback, bug reports, and feature requests are all welcome.

Quick Start

  1. Create a new project or open/import an existing .qse or .png.
  2. Paint in 2D or directly on the 3D model.
  3. Save the project as .qse.
  4. Export the final skin as .png (and .obj if needed).

Core Controls

  • Q Pencil
  • E Eraser
  • F Fill
  • M Box Select
  • I Eyedropper
  • W Zoom Tool
  • Space + drag to pan 2D canvas / rotate 3D viewport
  • Shift + drag in 3D viewport to pan
  • Mouse wheel in 3D viewport to zoom
  • Cmd/Ctrl + Z Undo
  • Cmd/Ctrl + Shift + Z Redo
  • Cmd/Ctrl + K Keybinds

Community Build Notes

  • Unsigned macOS builds may require right click Open on first launch.
  • Unsigned Windows builds may show SmartScreen warnings.
